## Propósito

Estandarizar la forma en la que los equipos de desarrollo ponen en práctica el proceso de Grooming
al dividir historias de usuario para su estimación

**¿Para qué sirve el Backlog Grooming?**
- Eliminar historias de usuario que ya no son relevantes
- Reordenar la prioridad de las historias de usuario
- Añadir o corregir estimaciones
- Dividir historias de usuario

## Entradas

- Backlog (PVG) priorizado
