## Propósito
Registrar en un documento el compromiso de las partes interesadas en un mutuo acuerdo para generar confianza en su ejecución.

## Entradas
Todo lo necesario para presentar el compromiso o negociar.
PVG
WBS
Plan de entregas
Plan de desarrollo
SRS
Políticas

## Descripción
| Fases | Actividades | Prácticas asociadas al CMMI |
|--------------|----|--------------|
| Identificar el Interés | El Team Member define el propósito del acuerdo a realizar. Identificar a los interesados. | PP 3.3 |
| Identificar artefactos | El Team Member prepara los artefactos que aporten valor al compromiso para incluirlos en el documento. | PP 3.1 |
| Asesoramiento grupal de artefactos | Se presentan los artefactos a utilizar que realmente aporten valor y se discuten grupalmente entre los interesados del equipo. | PP 3.2 |
| Redacción | Se redacta el documento con la información necesaria para el acuerdo. | PP 3.2 |
| Reunión | Reunión con los interesados. |  |
| Discusión | Se presentan los componentes que estarán implicados en el compromiso, dando apertura a la discusión del mismo. |  |
| Negociación | De ser necesario, se negociarán las partes del acuerdo en las que algún interesado no esté de acuerdo. |  |
| Edición | En caso de ser necesario, si la negociación provocó un cambio en el compromiso, se deberá actualizar el documento para su firma. | PP 3.2 |
| Firma | Firma de todos los interesados. | PP 3.2 |

## Salidas
Documento de compromiso firmado
