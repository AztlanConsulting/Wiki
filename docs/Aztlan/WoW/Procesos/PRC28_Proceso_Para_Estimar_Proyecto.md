## Propósito
Ejecutar el modelo generado de estimación del proceso anterior para sacar las estimaciones a nivel proyecto.

## Entradas
El modelo razonado de análisis a nivel proyecto y las estimaciones de los productos de trabajo/tareas del 1.2.

## Descripción
| Fases | Actividades | Prácticas asociadas al CMMI |
|--------------|--------------|--------------|
| Preparación |  Tener a la mano las estimaciones de las tareas/productos de trabajo individuales  |  1.2  |
| Consideración de Overhead en el Esfuerzo | Realizar un análisis de las implicaciones en el esfuerzo de las tareas. Puedes usar la guía [- Guía para consideraciones de overhead](../Guías/GUI10_Considerar_Proyecto_Overhead.md)  |  1.4  |  
| Integración |  Incluye las consideraciones de overhead como parámetros en el mismo modelo definifo, dado que estas consideraciones no fueron integradas a nivel tarea al momento de hacer estimaciones en 1.2   |  1.2  | 
| Ejecución |  Ejecuta el modelo/función ya con las implicaciones incluidas para sacar el esfuerzo/costo más huerístico del proyecto completo  |  ...  | 
| Documentación |  Guarda y documenta el resultado en un lugar accesible para el equipo y departamento  |  ...  |  

## Salidas
Estimaciones integrales y basadas en razón del esfuerzo/coste total del proyecto
