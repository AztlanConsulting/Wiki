## Propósito
Informar y ejecutar el modelo razonado de estimación del esfuerzo/costo del proyecto entero.

- PVG actividades

## Descripción
| Fases | Actividades | Prácticas asociadas al CMMI |
|--------------|--------------|--------------|
| Consideración de Overhead en el Esfuerzo | Realizar un análisis de las implicaciones en el esfuerzo de las tareas. Puedes usar la guía [- Guía para consideraciones de overhead](../Guías/GUI10_Considerar_Proyecto_Overhead.md)  |  PP 1.4, REQM 1.3  |  
| Integración |  Incluir las consideraciones de overhead como parámetros en el mismo modelo definido, dado que estas consideraciones no fueron integradas a nivel tarea al momento de hacer estimaciones en   |  PP 1.2, PP 1.3, PP 1.4  | 
| Ejecución |  Ejecutar el modelo/función ya con las implicaciones incluidas para sacar el esfuerzo/costo más huerístico del proyecto completo  |  PP 1.4  | 
| Documentación |  Guardar y documentar el resultado en un lugar accesible para el equipo y departamento (PVG)  |  PP 2.7  |  

## Salidas
Costo total documentado en el PVG
