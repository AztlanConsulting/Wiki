## Propósito
Regular el comportamiento, establecer direcciones y principios para los miembros del departamento.

## Entradas
Marcos éticos:
 - Dignidad humana.
 - Derechos humanos.
 - Principios éticos.
 - Valores éticos.
 
Comportamiento documentado del departamento.

## Descripción
| Fases | Actividades | Prácticas asociadas al CMMI |
|--------------|--------------|--------------|
| Identificación. | Identificar la consecuencia que tiene el comportamiento dentro del departamento, es decir, por qué estamos haciendo la política |    |
| Investigación. |  Generar propuesta de política ligada a un elemento del marco ético  |    | 
| Ejecución. | Detallar la política en la wiki según corresponda |    | 
| Validación. |  Validar con los miembros del departamento la adopción de la política, para ser aceptada, se ocupa mayoría absoluta de votos |    |  
| Documentación. | Realizar la actualización en la wiki si fue aceptada   |    |  

## Salidas
Política documentada en la wiki.
