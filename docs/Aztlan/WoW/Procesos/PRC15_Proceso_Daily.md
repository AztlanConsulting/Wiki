## Propósito

Dar seguimiento a las tareas realizadas e impedimentos de cada uno
de los miembros del equipo

## Entradas

- Backlog (PVG)

## Descripción

| Pasos         | Actividades                                                   | Prácticas asociadas al CMMI |
|---------------|---------------------------------------------------------------|-----------------------------|
| Ejecución     | El TL consulta la [plantilla de la Daily](../Plantillas/Plantilla_Daily.md)                   | PP.3.1 PP.2.1                      |
| Ejecución     | El TL abre el PVG del equipo y muestra el Tablero             | PP.3.1 RM.1.4                      |
| Ejecución     | El TL compara el progreso reportado por los TMs con las tareas planeadas en el PVG para verificar que el trabajo siga alineado con el plan del Sprint            | PP.3.1                             |
| Ejecución     | El TL hace las preguntas de la plantilla a cada uno de los TM | PP.3.1                      |
| Ejecución     | El TL identifica riesgos, bloqueos o dependencias mencionadas | PP.3.1 PP.2.2 RM.1.5               |
| Ejecución     | Adicionalmente se pueden hacer más preguntas si es pertinente | PP.3.1 PP.2.2                      |
| Documentación | El TL documenta las respuestas                       |                       |


## Salidas

- Registro de Daily
