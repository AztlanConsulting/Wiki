## Propósito
Establecer que requisitos hay para los datos y cómo son moldeados para el proyecto

## Entradas
* Requisitos de privacidad
* Requisitos de seguridad
* Listado de datos del proyecto que se debe recopilar
* Lista de datos gestionados
* Descripción del contenido y formato de los datos
