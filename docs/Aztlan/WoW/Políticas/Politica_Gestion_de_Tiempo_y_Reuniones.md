## Propósito
Establecer lineamientos formales para la planificación, ejecución y control de reuniones, presentaciones y uso del tiempo, garantizando eficiencia operativa y alineación organizacional.

## Áreas de proceso y prácticas en las que se aplica
- PP - Proyect Planning

## Política

1. Toda reunión deberá contar con:
- Objetivo definido.
- Agenda estructurada.
- Tiempos asignados por tema.
- Moderador responsable del control del tiempo.

2. Se establecerán bloques exclusivos para preparación de presentaciones, durante las cuales se suspenderán las actividades técnicas.

3. Las reuniones deberán programarse prioritariamente dentro del horario oficial de clase (15:00-20:00) o bloque alternativo (10:00-13:00).

4. Se declarará estado crítico cuando:
- Exista un retraso mayor al 20%.
- Riesgo alto sin mitigación.
- Bloqueo técnico mayor a 3 días.


Activándose un plan formal de contingencia.
