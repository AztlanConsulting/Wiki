# Consideraciones de datos o modelos relevantes
# Ejemplos a considerar en la búsqueda de información de proyectos pasados relevantes para el proyecto actual

## Propósito
Esta guía explica cómo determinar qué datos y/o modelos pasados son relevantes al proyecto actual, para fines de comenzar el proceso para crear un análisis razonado de estimación.

---

## Desarrollo

### Paso 1: Determinar características de proyecto actual por diferentes áreas
Estas son algunas áreas en que se agrupan características claves de un proyecto:
1. Tipo de organización
a. Empresa privada
b. Iniciativa de gobierno
c. Organización sin fines de lucro
2. Ámbito del proyecto
a. Construcción/infraestructura
b. Servicios
c. Comercio
d. Deportivo
e. Otros
3. Habilidades requeridas
a. Informáticas/software
b. Administración/negocios
c. Leyes/derecho
d. Infraestructura (ingeniería civil y similares)
e. Otros
