# ¿Qué es y cómo generar una matriz RACI?

## Propósito
Aprender a generar una matriz RACI

---

## Desarrollo

La matriz RACI es una forma de identificar los roles y responsabilidades dentro de un equipo para cualquier tarea.

- **R**: Responsable


Persona encargada de *realizar* el trabajo

- **A**: Accountable


Persona encargada de *supervisar* la finalización de la tarea

- **C**: Consultado 


Persona(s) que deben revisar y dar el visto bueno al trabajo

- **I**: Informado


Persona(s) que se les informa el progreso y cuándo es finalizado el trabajo


### ¿Cómo crear una matriz RACI?

1. Identifica los roles del proyecto
2. Detalla las tareas y entregables
3. Asigna los roles
4. Comparte la matriz de responsabilidades con el equipo
