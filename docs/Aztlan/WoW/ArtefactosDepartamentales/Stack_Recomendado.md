# Stack Recomendado

## Propósito
Esto ayuda mucho a su objetivo de que equipos se ayuden.

## Estructura

- HTML
- React
- EJS *(Default)*

## Estilo

- Tailwind
- Bulma
- Bootstrap *(Bootstrap icons)*
- CSS Normal *(Default)*

## Base de datos

- MongoDB
- Postgres SQL
- MariaDB SQL *(Default)*

## Framework

- FastAPI *(si es algo más con ciencia de datos)*
- NextJS *(facilita ruteo con React)*
- SpringBoot *(por si quieren Java)*
- Express *(Default)*
