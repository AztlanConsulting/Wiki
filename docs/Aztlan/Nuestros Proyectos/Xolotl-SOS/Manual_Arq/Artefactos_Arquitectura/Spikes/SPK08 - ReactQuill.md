**Duración:** 3-4 horas   
**Funcionalidad:** Crear un editor de texto enriquecido dentro de una app de React. 

## Beneficios

- Sencillez para el usuario final.
- Librería fácil de integrar si se programa desde cero la interfaz.
- Permite modificar diferentes tipos de formato.

## Desventajas

- Usa HTML, se debe usar una librería como DOMPurify para mostrar datos de forma segura.
- Como el anterior, tiene una cierta curva de aprendizaje que hay que superar.
- Puede ser redundante si se usa una interfaz de otra herramienta (como Payload), en lugar de programarla desde cero.

## Facilidad de aprendizaje

**Fácil**
