**Duración:** 1 día  
**Funcionalidad:** Genera póster automáticamente de los perros perdidos utilizando html / react  

## Beneficios

- Facilidad de uso e instalación
- Crear el póster en html
- Modificarlo a blanco y negro facilmente


## Desventajas

- Tiene que ser un objeto en la página
- Dificultad para manejar su estado

## Facilidad de aprendizaje

**Fácil**
