**Duración:** 1-2 días  
**Funcionalidad:** Base de datos NoSQL orientada a documentos  

## Beneficios

- Esquema flexible
- Diseño escalable
- Rápido en desarrollo
- Manejo de datos complejos


## Desventajas

- Menos consistente que SQL 
- No ideal para relaciones complejas
- Posible duplicación de datos
- Requiere buen diseño


## Facilidad de aprendizaje

**Fácil**
