**Duración:** 3-4 horas  
**Funcionalidad:** Servicio de geolocalización basado en IP.  

## Beneficios

- El plan gratuito presenta hasta 30,000 requests por mes.
- Incluye el HTTPS en el plan gratuito.
- Proporciona bastante información útil del usuario.
- Requiere una API Key pero es fácil de registrar.
- Compatible con REACT.
- Utilizado en grandes empresas.
- Retorna datos en formatos JSON y XML.
- En el plan pagado presenta una opción de que tan segura es la  IP de la persona que está en tu página.
- Presenta alta documentación.
- Detecta VPNs y Proxies.
- Sus servidores presentan un uptime de 99.9%.
- El plan gratuito ofrece el tipo de moneda.
- Tiene la opción de traducción de monedas tomando como base USD.
- Presentan un buen soporte técnico.


## Desventajas

- Plan gratuito limitado para producción.
- Presenta respuestas más pesadas que Ipapi.co.
- Registro de API Key obligatorio.
- En caso de superar el plan gratuito puede empezar a tener costo.
- El precio puede llegar a ser bastante costoso.
- Puede llegar a cometer errores.


## Facilidad de aprendizaje

**Fácil**
