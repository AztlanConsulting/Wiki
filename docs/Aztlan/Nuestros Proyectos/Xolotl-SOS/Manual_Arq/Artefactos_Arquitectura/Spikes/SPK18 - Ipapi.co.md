**Duración:** 2-3 horas  
**Funcionalidad:** Servicio de geolocalización basado en IP. 

## Beneficios

- Fácil y rápido de implementar.
- No requiere configuración compleja.
- Proporciona bastante información útil desde el país hasta el tipo de moneda.
- No requiere API Key para el plan gratuito.
- Funciona de manera automática sin necesidad de pedir permiso al usuario.
- Ofrece un plan customizable para tus necesidades.
- Compatible con REACT.
- Presenta un endpoint JSON sencillo, lo cual lo hace muy fácil de utilizar.
- Es utilizado por empresas grandes como Amazon.
- Retorna datos en diferentes formatos JSON, XML, CSV, YAML...
- Es demasiado fácil de integrar con Node.js.


## Desventajas

- Límite de 1,000 requests por día para el plan gratuito.
- HTTPS solo disponible para el plan de pago.
- La precisión depende de la IP y las VPNS pueden dar resultados incorrectos.
- Dependencia externa a sus servidores.
- Puede ser escalable, pero ya empieza a tener un costo.
- Es la menos estable de todas las opciones presentes.
