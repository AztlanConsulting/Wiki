**Duración:** 2-3 horas  
**Funcionalidad:** Creación de contenido para el blog

## Beneficios

- Compatibilidad con React.
- Conexión a cualquier base de datos a través de sus APIs.
- Facilidad de edición para perfiles no técnicos (WYSIWYG).
- Seguimiento y control de versiones sencillo en Git, porque está muy basado en código.

## Desventajas

- En comparación a otras alternativas, se debe usar más código que UI para editar datos.
- Tiene una cierta curva de aprendizaje que no está presente en un competidor como Strapi.

## Facilidad de aprendizaje

**Fácil**
