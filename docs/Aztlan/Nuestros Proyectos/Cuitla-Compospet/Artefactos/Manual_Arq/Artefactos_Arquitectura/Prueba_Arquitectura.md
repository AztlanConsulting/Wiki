### Propósito 
Implementar, probar y desplegar la API del backend.

### Arquitectura

### Prueba 1 - Express & NodeJS

Se realizó la implementación de node.js y express para la creación de la API para el backend, se realizaron peticiones a la API para verificar su correcto funcionamiento. A su vez, se desplego el proyecto en el servidor de Compospet.
