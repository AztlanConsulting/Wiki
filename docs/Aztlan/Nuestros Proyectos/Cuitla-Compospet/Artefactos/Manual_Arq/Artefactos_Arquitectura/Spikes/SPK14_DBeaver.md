**Duración:** 1 día  
**Funcionalidad:** Administrador de Base de Datos

DBeaver es una herramienta de administración de bases de datos que permite conectarse de forma visual a motores de base de datos como PostgreSQL, MySQL, MariaDB, SQLite y otros. En el proyecto nos servirá para administrar la base de datos del servidor sin depender únicamente de la terminal de PostgreSQL.  

## Beneficios
- Interfaz de usuario visualmente robusta
- Facilita el manejo de la base de datos a traves de la interfaz gráfica 
- Facilita ejecutar consultas SQL, navegar tablas, revisar esquemas y administrar la base de datos desde una sola herramienta
- DBeaver Community es una Open Source gratis
- Incluye resaltado de sintaxis, autocompletado inteligente de código
-  Migración de datos a diferentes formatos como (CSV, Excel, SQL, JSON)

## Desventajas

- Alto consumo de recursos
- Puede bloquear tablas por consultas que no se liberen
